# Scrap Companion releases

This public repository contains only the signed alpha update manifest for ScrapLink
testers. It does not contain the private Windows installer, API keys, invite codes,
player data, or the private release-signing key.

`latest.json` is verified by the Scrap desktop app before any package is accepted.
The installer is delivered from `scrap-link.com` only after a valid ScrapLink account
session is confirmed.
